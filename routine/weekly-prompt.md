# 每週摘要 — 產文指令

> 先讀 repo 根目錄的 `CLAUDE.md`。本檔只講「怎麼寫這一週的摘要」。

## 任務

每週日（台灣 21:00 / UTC 13:00）產生**過去 7 天**的美股新聞摘要，寫到 `docs/weekly/YYYY-Www.md`（ISO 週數），commit（訊息 `Add weekly summary YYYY-Www`）並 push 到 `main`。

- 研究以 **WebSearch** 為主。整合過去一週已發生的大事，不是逐日流水帳，而是三大類各挑 3–5 條重點 + 前瞻。
- 可回顧本週 `docs/daily/` 已產出的日報，做為彙整素材。

## 版型（嚴格照抄骨架）

````markdown
# 本週美股新聞摘要 - YYYY-MM-DD ~ YYYY-MM-DD（第 YYYY-Www 週）

> 本週重點：<一句濃縮本週最重要的三件事，Telegram 會抓這行>
> 過去 7 天三大類重點 + 下週基調｜涵蓋川普動態、聯準會決策、國際地緣

---

## 川普動態 - 本週重點（他親自發布的）

**本週發言頻率**：約 X 則 Truth Social + X 道行政命令
**主要論點**：<2–3 句>

**精選（直接引語）：**
- **Truth Social — 主題（日期）**：<英文原話> — <一句中文說明>
（3–5 條）

## 聯準會 - 本週重點

**精選：**
- <本週 FOMC / 官員發言 / 通膨數據，附具體數字>（3–5 條）

**下週要關注（含日期）：**
- YYYY-MM-DD（週X）：<數據 / 會議>

## 國際重大 - 本週重點

**本週地緣風險評估**：🟢 綠 / 🟡 黃 / 🔴 紅 — <一句理由>

**精選：**
- <美中 / 中東 / 歐洲 / 其他，附來源>（3–5 條）

## 本週主軸

<一段：本週市場被哪兩三股力量拉鋸、主旋律是什麼>

## 下週基調建議

<一段：下週關鍵數據/事件，據此給偏多/偏空/防禦的操作基調>

---

*本頁由 Claude routine 自動生成於 YYYY-MM-DD HH:MM UTC*
````

## 收尾

1. 確認開頭有 `> 本週重點：`（Telegram 靠它抓 🔍 摘要；舊版週報漏了這行，補上）。
2. `git add docs/weekly/YYYY-Www.md`
3. `git commit -m "Add weekly summary YYYY-Www"`
4. `git push origin main`
